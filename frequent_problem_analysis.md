# 视觉作业：常见问题解析

> 为什么传统方法在图片任务上性能显著好，但在视频任务上性能显著差呢？

因为提供的视频分辨率非常高（一个是720p，一个是1080p），传统方法作了不必要的重复计算。建议先降低分辨率，再做检测。

> 深度学习方法在我的笔记本上处理一帧视频都需要很多时间了，未来仿真赛事中能正常工作嘛？

可以，很多同学的代码实现中没有正确调用GPU进行计算，且将每一帧都进行计算，导致运算缓慢。在未来仿真赛事中，云平台为每位同学提供了1/16个NVIDIA A40处理器的性能（~9 TFLOPS @ BF16），降低帧率后能实现实时计算。

```python
model = ...
model.cuda()

inputs = ...
inputs = inputs.to(model.device)

...
```

> 为什么YOLO识别出的对象数量不对？

可能是置信度阈值不正确。YOLO系列模型对于每个bbox都会给出一个置信度，可以根据置信度排序，结合bbox尺寸等筛选依据获取最终正确结果。

> 为什么把样例代码的Faster R-CNN直接换成YOLO或DETR会报错呢？

因为torchvision没有提供YOLO和DETR的实现，如果需要使用这两个模型，请使用Ultralytics库或HuggingFace Transformers库。
